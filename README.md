# Go Solving App Repository

This repository contains the source code for the Go Solving App, a cross-platform mobile application for Go (Weiqi) board recognition and analysis.

## Structure

- **[go_solving_app](./go_solving_app/)**: The main Flutter application. See its [README](./go_solving_app/README.md) for detailed setup and usage instructions.

## Getting Started

To get started with the mobile app:

```bash
cd go_solving_app
flutter pub get
flutter run
```

## Features

- Board Recognition using OpenCV
- AI Analysis with KataGo
- Cross-platform (Android/iOS)
