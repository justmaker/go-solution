# Go Solving Project

This repository contains a mobile application for analyzing Go (Weiqi) board positions.

## Projects

- **[Go Solving App](go_solving_app/)**: A Flutter-based cross-platform app (Android/iOS) that uses OpenCV for board recognition and KataGo AI for move analysis.
